# Installation and setup instruction
Clone down this repository. You will need [node](https://nodejs.org/en) and [npm](https://www.npmjs.com/) installed globally on your machine.

Installation : 
 ` npm install`

To run locally :
 `npm run dev`

Visit app : 
 https://localhost:3000
